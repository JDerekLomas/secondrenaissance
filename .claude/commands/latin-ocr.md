# Latin OCR

Transcribe Renaissance Latin text from the image at: $ARGUMENTS

## Instructions

1. Read the image file using the Read tool
2. Transcribe the Latin text with these guidelines:
   - Start with `[[notes: ...]]` describing page condition, layout, damage
   - Preserve original capitalization, spacing, line breaks
   - Use Markdown to mirror source layout (headings, centered text, italics)
   - Mark uncertain characters with `[[?reading]]`
   - Expand abbreviations only when certain; otherwise note ambiguity
   - Include `[[page: N]]` if visible
3. Output the transcription in a code block
